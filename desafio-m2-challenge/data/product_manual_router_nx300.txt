================================================================================
OVERVIEW
================================================================================

The Router NX300 is TechStore Plus's premium Wi-Fi 6 (802.11ax) wireless router.
It delivers combined speeds of up to 3000 Mbps (574 Mbps on 2.4 GHz + 2402 Mbps
on 5 GHz), covers up to 2,500 sq ft as a standalone unit, and supports up to 64
simultaneous connected devices. Four Gigabit Ethernet LAN ports and one Gigabit
WAN port are provided for wired connections.

Security: WPA3-Personal, SPI firewall, automatic firmware updates (opt-in).
Management: web UI at 192.168.1.1 or NX300 mobile app (iOS / Android).

UPGRADE PATH: The Router NX300 (2024) replaces the Router NX200 (2022). The NX300
delivers Wi-Fi 6 speeds approximately 40% faster than the NX200 and adds WPA3
security and full compatibility with the Smart Hub Home mesh network feature.
Existing Router NX200 owners are eligible for a trade-in credit of $50 toward the
Router NX300 through the TechStore Plus Trade-In Program (see trade_in_program.txt).

COMPONENT HIGHLIGHTS
================================================================================

The Router NX300 is powered by enterprise-grade networking silicon:

- Processor: Qualcomm IPQ5018 dual-core ARM Cortex-A53 SoC at 1.0 GHz.
  The IPQ5018 integrates a dedicated Wi-Fi 6 radio controller, hardware NAT
  acceleration, and a built-in security processor — reducing latency for heavy
  workloads such as 4K streaming and video conferencing.
- RAM: 512 MB DDR3 (not user-upgradeable).
- Flash: 128 MB NAND (firmware storage).
- Wireless radios: 2x integrated 2x2 MIMO antennas (2.4 GHz + 5 GHz dual-band).
- Ethernet switch: 5-port Gigabit (1 WAN + 4 LAN).

COMPATIBLE ACCESSORIES & ECOSYSTEM INTEGRATION
================================================================================

The Router NX300 is the recommended networking backbone for the TechStore Plus
smart home and home office ecosystem. Key integrations:

1. Smart Hub Home (SKU: SH-HOME-2023) — Mesh Network Partner
   The Smart Hub Home requires a Router NX300 or Router NX200 to enable its mesh
   networking feature. When the Smart Hub Home is connected to a Router NX300 via
   Ethernet, the two devices form a coordinated mesh network that extends Wi-Fi
   coverage to up to 4,000 sq ft total. The Smart Hub Home acts as a secondary
   access point, and the Router NX300 acts as the mesh gateway controller.
   Without a Router NX300 or Router NX200, the Smart Hub Home operates only as a
   standalone smart device controller (Zigbee/Z-Wave hub) — the mesh Wi-Fi
   extension feature is unavailable.

2. Smart Cam 4K (SKU: SM-CAM-4K) — Recommended Pairing
   The Smart Cam 4K works with the Router NX300 for local on-network streaming at
   full 4K resolution. When the Smart Cam 4K is connected to a network managed by
   the Router NX300 (either directly or via the Smart Hub Home mesh), local stream
   latency is under 200 ms. Cloud streaming is also supported, but local streaming
   via the Router NX300's LAN is recommended for privacy and performance.

3. Laptop Pro X1 (SKU: LP-X1-2024) — Wi-Fi 6 Client
   The Laptop Pro X1's Intel AX211 module supports Wi-Fi 6, making it a natural
   partner for the Router NX300. Using both devices on the same network delivers
   up to 1.2 Gbps throughput over Wi-Fi 6 (vs. ~400 Mbps with Wi-Fi 5 routers).
   This pairing is included in the Home Office Bundle recommendation.

4. WorkStation Z (SKU: WS-Z-2024) — Wired Ethernet Recommended
   The WorkStation Z desktop connects to the Router NX300 via one of the four
   Gigabit Ethernet LAN ports for maximum bandwidth. Wired connection is preferred
   for WorkStation Z to support high-volume data transfers.

MESH NETWORK SETUP WITH SMART HUB HOME
================================================================================

To create a mesh network using the Router NX300 and Smart Hub Home:

1. Complete the standard Router NX300 setup (see Initial Setup below).
2. Connect the Smart Hub Home to a LAN port on the Router NX300 via Ethernet.
3. In the NX300 mobile app, go to Network > Mesh > Add Mesh Node.
4. The app will automatically detect the Smart Hub Home and pair it as a secondary
   access point. The Smart Hub Home extends the NX300 network under the same SSID.
5. Devices throughout your home will automatically roam between the Router NX300
   and the Smart Hub Home based on signal strength.

Note: The Router NX200 also supports the Smart Hub Home mesh pairing, but maximum
mesh throughput is lower than with the Router NX300 due to the NX200's Wi-Fi 5
(802.11ac) limitation.

INITIAL SETUP
================================================================================

1. Unbox the router and place it in a central, elevated location for optimal
   coverage. Avoid enclosures or proximity to microwaves and cordless phones.
2. Connect the WAN (internet) port (blue) to your modem using the supplied
   Ethernet cable.
3. Connect the power adapter and press the Power button on the back panel.
   Wait 60 seconds for the router to boot fully (solid white LED = ready).
4. From a device, connect to the default Wi-Fi network:
   - SSID: NX300_SETUP
   - Password: printed on the label on the router's bottom panel.
5. Open a browser and navigate to http://192.168.1.1 or http://nxrouter.local.
6. Follow the Setup Wizard:
   a. Set a new admin password (minimum 12 characters).
   b. Configure your Wi-Fi SSID and password for both 2.4 GHz and 5 GHz bands.
   c. Select your ISP connection type (Dynamic IP / Static IP / PPPoE).
   d. Enable automatic firmware updates when prompted.
7. Register your warranty at techstoreplus.com/warranty using the serial number
   on the bottom label (format: NX300-XXXXXXXX).

================================================================================
TROUBLESHOOTING
================================================================================

ISSUE: Router will not connect to the internet (WAN LED red or off)
- Verify the Ethernet cable is firmly seated in both the modem WAN port and the
  router WAN port.
- Power-cycle the modem first (unplug 30 s), then power-cycle the router.
- Check your ISP connection type in the Setup Wizard — PPPoE requires a username
  and password supplied by your ISP.
- See also: support_router_wont_connect.txt for a detailed step-by-step guide.

ISSUE: Smart Hub Home mesh not forming after pairing
- Confirm the Smart Hub Home is connected by Ethernet to a LAN port, not the WAN
  port (blue).
- In the NX300 app, go to Network > Mesh > Refresh. If the Smart Hub Home still
  does not appear, power-cycle the Smart Hub Home (hold its side button 5 s).
- Verify both devices are running current firmware (NX300 app: Settings > Firmware).

ISSUE: Wi-Fi signal weak in certain rooms
- Move the router to a more central location or elevate it.
- Enable Beamforming in the Advanced Wireless settings (web UI > Wireless >
  Advanced > Beamforming = ON).
- For large homes, add the Smart Hub Home as a mesh node (see Mesh Network Setup).

ISSUE: One device cannot connect to Wi-Fi
- Forget the network on the device and reconnect.
- Reboot both the router and the affected device.
- Check that the device supports WPA3 or set the router to WPA2/WPA3 mixed mode
  (web UI > Wireless > Security > Mixed Mode).

ISSUE: Admin password forgotten
- Perform a factory reset: hold the Reset button on the back for 10 seconds until
  all LEDs flash. Default credentials are printed on the bottom label.
  WARNING: Factory reset erases all custom settings.
